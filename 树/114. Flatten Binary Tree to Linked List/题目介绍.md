
[二叉树转换成链表](https://leetcode.com/problems/flatten-binary-tree-to-linked-list/)

给定二叉树，变成只有右儿子的链表形式

For example, given the following tree:
```
    1
   / \
  2   5
 / \   \
3   4   6
```

The flattened tree should look like:
```
1
 \
  2
   \
    3
     \
      4
       \
        5
         \
          6
```
