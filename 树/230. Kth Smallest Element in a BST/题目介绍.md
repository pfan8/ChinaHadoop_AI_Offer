
[查找二叉搜索树BST的第k小元素](https://leetcode.com/problems/kth-smallest-element-in-a-bst/)

查找第k个小的元素

Example 1:
```
Input: root = [3,1,4,null,2], k = 1
   3
  / \
 1   4
  \
   2
Output: 1
```
Example 2:
```
Input: root = [5,3,6,2,4,null,null,1], k = 3
       5
      / \
     3   6
    / \
   2   4
  /
 1
Output: 3
```
